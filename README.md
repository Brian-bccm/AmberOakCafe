# Amber & Oak Cafe

Premium restaurant landing website built for a freelance portfolio project.

## Stack

- React
- Vite
- Tailwind CSS
- Netlify Forms
- WhatsApp reservation CTA

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm test
npm run lint
npm run build
```

## Deployment

Netlify build settings are configured in `netlify.toml`.

- Build command: `npm run build`
- Publish directory: `dist`
